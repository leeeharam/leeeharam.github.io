처음으로 풀어본 알고리즘 문제. 나중에 코드 수정하자

와 되게 쉬운 문제가 선린에 나왔네? 라고 생각했는데, 결코 쉽지는 않았다. 그렇다고 고난도의 코드를 사용하거나 엄청난 알고리즘 지식을 써야하는 문제도 아니라 골치가 더 아팠다.

이게 문제가 상당히 까다롭다. N - 1까지는 알겠는데 1병 남았으면 bottles 대신 bottle을 써야한다. if문 써서 1병이면 bottle로 바꾸면 되는거 아니야? 라고 생각 할 수 있겠지만 N -1에도 똑같이 적용을 해야하는데,

if (N - 1 == 1)

이걸로 해버리면 오류가 생겨버린다. 그래서 N이 1이면 그냥 정해진 값을 출력하는걸로 바꿨다. 그리고 no more로 바꾸는 부분에서도 정상적으로 되지 않아서 else로 다시 정해진 값을 출력했다. 근데 사실 저 부분 때문에 어차피 따로 만들어야 했다.

Go to the store and buy some more

계속 가다가 마지막에는 또 다르다;; 멍청하게 저걸 못 찾아서 저걸로만 10분 넘게 애를 먹었다. 입력값을 다시 변수에 넣어주는 작업도 필요했다.

var K2: String = K

bottles의 "s" 자리에 변수를 넣어 기본값을 s로 설정해주고, K가 1이면 ""를 넣어 s가 붙지 않게 해주었다.

if ((K.toInt() - 1) == 1) { S = "" }

그런데 이 부분은 if문 안의 부분이라 작동이 뭔가 안됐다. 그래서 S를 다시 선언 해 주었다.

else {
    println("No more bottles of beer on the wall, no more bottles of beer.")
    print("Go to the store and buy some more,")
    println(" ${K2} bottle${S2} of beer on the wall.")
}

전체 코드

import java.io.BufferedReader
import java.io.InputStreamReader
fun main(args: Array) {
    var br = BufferedReader(InputStreamReader(System.`in`))
    var K: String = br.readLine()
    var K2: String = K
    var S: String = "s"
    var S2: String = "s"

    for (i: Int in 1 .. K.toInt() + 1) {
    if (K.toInt() == 1) { S = "" }
    if (K2.toInt() == 1) { S2 = "" }

    if (K.toInt() > 0){
        print("${K.toInt()} bottle${S} of beer on the wall,")
        println(" ${K.toInt()} bottle${S} of beer.")

    if (K.toInt() > 1) {
    if ((K.toInt() - 1) == 1) { S = "" }
        print("Take one down and pass it around,")
        println(" ${K.toInt() - 1} bottle${S} of beer on the wall.")
    }
    else {
        print("Take one down and pass it around,")
        println(" no more bottles of beer on the wall.")
    }
    println()
    }
    else {
        println("No more bottles of beer on the wall, no more bottles of beer.")
        print("Go to the store and buy some more,")
        println(" ${K2} bottle${S2} of beer on the wall.")
    }
    K = (K.toInt() - 1).toString()
    }
}
